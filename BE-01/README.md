# BE-01 - Task Management API

## Overview

A simple Task Management REST API built using **Python** and **Flask**. This project implements CRUD (Create, Read, Update, Delete) operations using in-memory storage.

---

## Features

- GET /tasks
- GET /tasks/{id}
- POST /tasks
- PUT /tasks/{id}
- DELETE /tasks/{id}

---

## Tech Stack

- Python
- Flask

---

## Installation

```bash
pip install -r requirements.txt
```

---

## Run the Application

```bash
python app.py
```

The server starts at:

```
http://127.0.0.1:5000
```
